# Lore

This file defines the core world logic that supports the game concept.

It does not attempt to explain the entire world.

It establishes the minimum lore needed to make the vertical slice coherent and readable.

## Scope

This lore currently focuses on:

1. [World Scenario](./world.md)
2. [John Riggs / Lucky Smith](./riggs.md)
3. [The Dice](./die.md)
4. [The Rig](./rig.md)

## Core Premise

The world contains rare future-era artifacts and engineered devices that can influence probability through field effects.

These systems are not normal casino cheating tools.

They operate at the border of advanced mechanics, unstable energy engineering, and partially unexplained forces that many people treat as quasi-magical.

The player does not simply hack software.

The player uses illegal machines and forbidden components to interfere with luck, probability, and event outcomes in the physical world.

## Lore Function

The lore must support three things:

- why dice exist as real components inside the machine
- why different dice types can reinforce or disrupt each other
- why building and operating such rigs happens through hidden underground networks

## Tone

The tone should remain:

- neon-noir
- technical
- occult-adjacent
- illegal
- unstable
