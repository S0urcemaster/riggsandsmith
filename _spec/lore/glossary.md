# Glossary

This file stabilizes the meaning-bearing lore terms used across the specification.

Terms should be added here when they define world logic, setting meaning, device meaning, or the role of a person, object, or phenomenon in the world.

## Lucky Smith

Lucky Smith is the illegal identity used by John Riggs in his hidden rigging life.

It is not a separate person.

## Fate Dice

Fate dice are rare future-era probability units that generate localized fields which influence surrounding systems and event outcomes.

They are real in-world components rather than ordinary gambling dice.

## Rig

A rig is an illegal probability hacking machine built to host and drive fate dice.

It is a field manipulation platform that tunes, powers, and combines dice so their effects can be directed into a target system.

## Run

A run is an illegal attack against a target machine, especially a gambling machine, carried out through the use of a rig.

In world terms, it is the forbidden attempt to bend the target system toward distorted outcomes, profit, or disruption.
