# Game Loop

## Core Loop

The playable round begins when John Riggs gets home.

The core game loop is:

1. Prep
2. Run
3. Result

The next day begins off-screen and leads back into the next playable Prep phase.

## Prep

Prep is the home phase.

Here the player works on the rig, adjusts the build, and prepares the next illegal run.

## Run

Run is the active machine phase.

Here the player takes the prepared rig into the field and uses it against slot machines or similar targets.

## Result

Result is the consequence phase.

Here the run resolves into profit, loss, suspicion, damage, collapse, or police pressure that carries into the next cycle.
