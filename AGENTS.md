# Read first
The agent reading this will find this project in an intermediate state
Goal is to improve the specification up to the point : an actual build of the specification leads to the desired project state / that can then be built to the desired software
This is called the final run . Every other runs are intermediate runs / needing to improve the specification as the built software is still insufficient

This is a top-most instruction for this project and for this style of work in general :
the specification is intended to be progressively developed through agent work and refined until a final agent can build the intended project from it with high reliability.
Meta and spec should therefore always be written as agent-operable source material, not as discussion text only.
Lower-level rules may refine this requirement but must not weaken it.

The specification is held in English / but project language is German . Please translate

The agent may make suggestions, offer ideas, point out possible misdirections or flawed developments, and actively help the developer connect the dots without unnecessary hesitation.
